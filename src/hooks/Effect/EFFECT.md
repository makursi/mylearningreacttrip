# 什么是副作用函数?

1.  纯函数(非副作用函数):输入相同, 输出一定相同, 且不干"额外的事"

        function add(a, b) {
        return a + b; // 只计算，不干别的 → 纯函数 ✅
        }

    纯函数不会修改外部的状态, 也不依赖外部可变状态.
    **纯函数内部的操作不会影响外部的变量、文件、数据库等**

2.  副作用函数:
    # 副作用函数 指的是那些在执行时会改变外部状态或依赖外部可变状态的函数。

# 在编程中，副作用包括：

发网络请求（fetch）
修改 DOM
订阅事件（如监听键盘）
设置定时器（setTimeout）
修改全局变量

## useEffect 就是 React 组件里的“事后小助手” 它会在组件渲染完成后，帮你做一些“副作用”事情（比如发请求、改标题、监听事件等）。

## “副作用” = 和 UI 渲染无关，但又必须做的事
